Web Scraping Space Data
Using Python

In this project, I used Python libraries to scrape
Payload and Launch Vehicle data from Gunter's Space Page
saving me hours of manual data entry.

  • View Code

I obtained the raw data through a public dataset available on Gunter's Space Page.

The raw data consisted of space data with fields like Operators, Mass, Launch Organizations, Design Life, COSPAR IDs, etc.

I was given the task to update a database with daily space launches and add as many details (mass, operator, design life, etc.) as possible. The work became very mundane and monotonous that I decided to build a web scraping tool to automate my data collection process and leave it to me to handle the QC after the data has been scraped. This tool has helped me and my company reclaim budget hours and be more productive in my current role as a data analyst. I included a date range option so the end user can choose what day, week, month, or quarter, they want to scrape.
